FastHome

Plataforma moderna para compra, venda e locação de imóveis. Desenvolvida com Next.js, React, TypeScript e Supabase.

Ver Demo GitHub

Principais Funcionalidades

Busca Avançada

Sistema de busca inteligente com filtros por preço, localização, número de quartos e muito mais.

Mapas Interativos

Visualização de imóveis em mapas interativos usando Leaflet com geolocalização precisa.

Design Responsivo

Interface otimizada para dispositivos móveis, tablets e desktops com experiência fluida.

Autenticação Segura

Sistema de autenticação com Google OAuth, email/senha e recuperação de senha.

Upload de Imagens

Sistema de upload com compressão automática e armazenamento seguro no Supabase Storage.

Internacionalização

Suporte completo para português e inglês com next-i18next.
